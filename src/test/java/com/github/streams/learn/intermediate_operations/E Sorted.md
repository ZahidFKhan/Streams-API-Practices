# Sorted:

The `sorted` method is used to sort the stream.

```java
List<Integer> result = List.of(5, 2, 4, 1, 3).stream()
        .sorted()
        .toList();

System.out.println(result);
```

#### Output:

```java
[1, 2, 3, 4, 5]
```
